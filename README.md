# hunter_gather
An early-human tribe 'simulator' written in Lua using the LOVE2d framework

Commands:
- help
- close
- reset
- clear
- cut [ number ]
- night [ on | off ]
- catalog
- debug [ on | off ]
- treebug [ on | off ]
- waterbug [ on | off ]

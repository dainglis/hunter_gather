# hunter_gather
An early-human tribe 'simulator' written in Lua using the LOVE2d framework

Commands:
- help
- close
- reset
- clear
- cut [ %number ]
- night [ on | off ]
- catalog
- echo [ on | off | %message ]
- debug [ on | off ]
- treebug [ on | off ]
- waterbug [ on | off ]

Aliases:
- on        : true, enable, active, -t
- off       : false, disable, inactive, -f
- close     : quit, exit
- clear     : cls
- catalog   : catalogue
